# jenkins-x

Create Jenkins X cluster on Google Cloud
```
jx create cluster gke
```

Select the GCP project
- rising-timing-220806
- zone
- us-central1-f
- n1-standard-2
- Minimum Node 3
- Maximum Node 5
- Serverless Jenkins X Pipelines with Tekton


See the env
```
jx get env
```

quickstart
```
jx create quickstart
```
